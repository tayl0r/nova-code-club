# NOVA Code Club Flyer — Design

## Overview

Single-page HTML flyer for the NOVA Code Club at a middle school. Serves as both a printable hallway flyer and a landing page published to tsteil.com via GitHub Pages.

## Content

1. **Header** — "NOVA Code Club" with bold, techy/gaming aesthetic
2. **Hook** — "Build Your Own Web Game Using AI"
3. **Pitch** — Each student gets their own game project. No coding experience needed. Describe what you want, AI builds it live. Every week your game gets better.
4. **How It Works**
   - Write your game ideas in a Google Doc during the week
   - Each Friday, we bring your ideas to life in real-time
   - Collaborate with other students as your game evolves
   - After 8 weeks, you have a playable web game
5. **Details**
   - Grades 6-8
   - Fridays, 3:45 – 5:00 PM
   - 8 weeks (start date TBD)
   - No signup needed — just show up
6. **Footer** — "Powered by Claude Code Max"

## Visual Direction

- Dark background with bright accent colors (game-inspired)
- Bold, modern typography that appeals to middle schoolers
- Clean enough for parents to read

## Technical Approach

- Single `index.html` with inline CSS, no build tools
- Print stylesheet so it can be printed as a one-page hallway flyer
- GitHub Pages deployment to tsteil.com
- Public repo: `nova-code-club`

## Club Mechanics

- One laptop (club leader's) with Claude Code Max subscription
- Leader drives the keyboard, switching between student projects each session
- Students prepare prompts in personal Google Docs between sessions
- Leader pastes student prompt, then iterates in real-time with the student collaborating
- All students can watch and participate during each other's sessions
